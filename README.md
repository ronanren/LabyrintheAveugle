# LabyrintheAveugle

## Librairies

```bash
# for manjaro 
pamac install glfw-x11 freealut sdl2_image
# for debian
sudo apt install g++ mesa-utils libglew-dev libglfw3-dev libsdl2-dev libsdl2-image-dev libopenal-dev libalut-dev freeglut3-dev

make cleanalllibs && make main
```
